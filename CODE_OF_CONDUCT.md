# Code of Conduct

Don't be a jerk.

That covers 99% of it. Specifically:

- Be helpful in issues and PRs. People are here to build things.
- Critique code, not people.
- If someone's new, help them — we were all new once.
- No harassment, no slurs, no bad-faith arguments.

If something feels off, open a private issue or contact a maintainer. Reports are handled with discretion.
